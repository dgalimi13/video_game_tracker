#Description

This app is called video_game_tracker. It is meant to track a users collection of consoles and video games. A user can register, login and then add consoles and games accordingly. a user can only edit consoles or games that belong to him/her.

#Installation

You can fork and clone this app from https://github.com/dgalimi13/video_game_tracker/tree/master. once cloned run bundle install and then rake db:migrate. start your server by running shotgun and youll be able to use this app
